# Sentiment Analysis of U.S. Media Coverage 

## In collaboration with Deshan DeMel, Marian Khan, and Nahian Rashha

- - Built a text analytics pipeline to process and analyze unstructured news data, comparing how left-, center-, and right-leaning outlets frame U.S. immigration across election cycles
- - Extracted textual sentiment and language patterns and translated findings into insights and intuitive visuals on how narratives shift across sources and time
